**Question:** For part 2, can I run any of the tests locally without making a pull request?

**Answer:** You can run npm run test locally to run the tests. Also you can make a pull request to begin with and everytime you commit and push to the repo, the tests will be rerun on that pull request.

**Question:** I’m getting an error saying ‘jest’ is not recognized as an internal or external command.

**Answer:** Try running npm install to install the dependencies from the package.json file.

**Question:** I’m not able to push my empty folders to my GitHub repo .

**Answer:** Git doesn’t let you push empty directories. If you wish to create the folder structure for the lab beforehand, you can do so on your local machine or VSCode, but the folders will only show up on your repo after you start adding files to them and pushing to GitHub.
